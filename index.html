MIDI2LR

MIDI2LR is an application that translates MIDI Control Change commands into Lightroom actions. The wiki is read-only to prevent spamming. Contact me if you wish to add any pages or text.

Latest Release

You can always download the latest release from https://github.com/rsjaffe/MIDI2LR/releases/latest

Supported platforms

A build for Windows is currently available, but the underlying framework is cross-platform, so a Mac build can easily be made

Proposed Project Plan

Please contact me if you have suggestions/changes for the plan.

  1. Move conversion from LR range to controller range from lua code to c++ code. Goals: remove the unwanted fader motor action caused by rounding errors, and prepare the application for wide-range MIDI control codes.
  2. Implement NRPN MIDI message handling. This will help with the wide-range parameters such as Color Temperature, as sensitivity of control can then be altered by speed of movement.
  3. Move on to other major improvements—order not yet determined.
