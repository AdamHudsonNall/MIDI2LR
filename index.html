MIDI2LR

An application and plugin to remotely control Lightroom 6 with a MIDI controller

Windows (.zip) Mac (.zip) View on GitHub

Supported platforms

  • Windows 7/8/10 64-bit
  • Mac OS X 10.9+ 64-bit

Prerequisites

  • Lightroom 6+
  • Any USB MIDI controller (eg: Korg nanoKONTROL, etc.)

Demo



Getting Started

  1. In Lightroom, add the MIDI2LR plugin by going to File>Plugin-Manager>Add and navigating to and selecting the MIDI2LR.lrplugin folder

  2. With a MIDI controller connected, launch the MIDI2LR application (should automatically launch with v0.3.0+)
    • On OSX, it might be necessary to Ctrl-click the MIDI2LR app and select 'open'.
    • Alternatively, go into System Preferences>Security & Privacy> and allow Apps from Anywhere, or individually allow MIDI2LR
  3. Moving any controllers will populate the mapping table and allow you to set functions. You can also save/load mappings.


Updating a Previous Version

  1. Close the MIDI2LR application if running
  2. In Lightroom, remove the old MIDI2LR plugin by going to File>Plugin-Manager>Remove
  3. Add the new MIDI2LR plugin by going to File>Plugin-Manager>Add and navigating to and selecting the MIDI2LR.lrplugin folder
  4. Both the plugin and application should display the same version
